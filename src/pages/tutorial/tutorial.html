Welcome to Electromanager!

The following slides will serve as a quick guide for using the app.

Swipe to next slide.

Create Category

Click on the Create Category button to create a new category.

Add Appliance

Click on the icon to add a new appliance to the category.

Edit And Delete

Click on a category name to edit its name.

Click on an appliance to edit its details.

Swipe to delete a category or appliance.

Catalogue And Menu

Click on to open the catalogue page.

Click on to open the menu tab

Saved Appliances

All added appliances are stored under the saved appliances page.

Appliances in the list can be edited on click and deleted on swipe.

Saved appliances can be used to automatically fill up appliance details.

Variable Power Consumption

Enable the Variable Power Consumption option when adding an appliance which does not consume a constant amount of power, then select the type of appliance you are adding from the drop down menu which appears.

Appliance Calibrator

The appliance calibrator in the reports page can be used to get instant cost and kWh consumption readings.

Tariff, Tax And Flat Rate

Input the kWh range and cost to add a new Tariff Rate range. Click on Clear List to delete all entries.

Input your tax rate and electric bill flat fee to add them into the summary page bill calculation.

Make sure to check the help page for any further details.

That's all!
